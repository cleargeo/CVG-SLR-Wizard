# CVG SLR Wizard — Tools

> © Clearview Geographic LLC | Internal Use Only

Utility scripts and helper tools for the CVG SLR Wizard.

## Planned Tools

- `expand_station_table.py` — Script to expand RSLR table to all 75 TR-083 stations
- `validate_dem.py` — DEM integrity checker (nodata %, vertical accuracy check)
- `batch_report_gen.py` — Generate reports for multiple existing output JSON files
- `fetch_all_stations.py` — Bulk fetch NOAA CO-OPS station metadata
