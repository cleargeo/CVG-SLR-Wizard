Core Capabilities

NOAA-Standard SLR Projections for Coastal Planning

The CVG SLR Wizard delivers peer-reviewed, NOAA Technical Report 083 sea level rise projections in seconds — fully compatible with FEMA Risk MAP and NOAA coastal resilience workflows.

📡

NOAA TR-083 (Sweet et al. 2022)

All six official NOAA/IPCC-aligned scenarios: Low, Intermediate-Low, Intermediate, Intermediate-High, High, and Extreme — the definitive U.S. federal standard for coastal planning.

📍

Station-Based Local Projections

Site-specific projections relative to 500+ NOAA tide gauge stations nationwide. Account for local subsidence, vertical land motion, and regional ocean dynamics automatically.

📊

Sensitivity Analysis

Instantly compare all six scenarios for your station and target year in a single table — critical for risk-based planning, vulnerability assessments, and adaptation strategy development.

🔗

Storm Surge Wizard Integration

SLR projections pipe directly into the CVG Storm Surge Depth Grid Wizard — add future sea level to any storm surge water surface for seamless future-conditions inundation analysis.

📏

VDatum Datum Support

Report projections in NAVD88, MLLW, MSL, or any supported NOAA VDatum vertical datum. Essential for coordinating with FEMA, USGS, and NOAA deliverables.

🌐

REST API Access

Full OpenAPI/Swagger REST API for batch queries, GIS integration, and programmatic workflows. Ideal for automated vulnerability assessments across large coastal portfolios.
